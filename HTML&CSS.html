Conceitos iniciais -- HTML

Quando vamos estruturar uma página em HMTL é necessário respeitar alguns fundamentos e formatos:

  • !DOCTYPE html: Informa para página a versão do HTML.
  • head: Local que passamos informações para página.
  • body: Local onde contém o conteúdo da página.

Estruturando a página, ficaria da seguinte forma:

<!DOCTYPE html>
< html >
< head >
< /head >
< body >
< /body >
< /html >

Tags conhecidas..,

  • h: Usada para títulos
  • p: Usada para paragráfos
  • strong: Por palavras em negrito
  • em: Para por palavras em itálico

Aprimorando a estrutura...

<!DOCTYPE html>
< html lang="pt-br">
(configura o idioma da página)
< head >
< meta charset = "UTF-8 >
(configura os caracteres)
< title > tílulo < / title >
(configura o título da página/aba)
< body >
< / body >

Conceitor iniciais CSS

Quando vamos introduzir o css na nossa página, precisamos criar um novo arquivo terminado em .css e fazer a linkagem no head
< link rel = "stylesheet" href = "nome.css" >

Alguns configurações do css:

  • text-align: alinha o texto dentro da box. (center: alinha o texto no centro da box)
  • font-size: define tamanho da fonte.
  • width: define a largura
